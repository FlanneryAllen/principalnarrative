# Messaging Pillars

## Primary Messaging Pillars

### Pillar 1: Instant, Intelligent Review

* **Core Message**: CodePilot delivers AI-powered code review in under 120 seconds, catching issues before they become problems.
* **Supporting Points**:
  * First review comment in \&lt;120  seconds (p95)
  * Analyzes security, performance, style, and logic
  * Learns your team's patterns and preferences
* **Proof Points**: \[perf-001, perf-002, adopt-001]
* **Use When**: Speed/efficiency conversations, competitor comparisons, developer productivity discussions

### Pillar 2: Human-AI Teamwork

* **Core Message**: CodePilot amplifies your senior developers - it doesn't replace them. AI handles the routine so humans focus on architecture and mentorship.
* **Supporting Points**:
  * AI explains every suggestion with reasoning and context
  * Developers approve, modify, or dismiss suggestions
  * Team knowledge captured and shared automatically
* **Proof Points**: \[roi-001, quote-001, quote-003]
* **Use When**: Addressing "will AI replace reviewers" concerns, team dynamics, knowledge transfer

### Pillar 3: Enterprise-Grade Security

* **Core Message**: Your code never leaves your environment. CodePilot's self-hosted option keeps analysis inside your security boundary.
* **Supporting Points**:
  * SOC 2 Type II certified
  * Self-hosted deployment available
  * Zero code storage - stream analysis only
* **Proof Points**: \[proof of SOC 2 certification]
* **Use When**: Enterprise sales, security-conscious buyers, regulated industries

## Category Definition

**Category Name**: Intelligent Code Collaboration
**Category Description**: A new category of developer tools that combine AI analysis with human expertise to accelerate code review while maintaining quality. Unlike traditional review tools (passive) or AI coding assistants (active generation), intelligent code collaboration augments the review process itself.
**Why This Category**: Positions us between incumbents (GitHub, GitLab built-in review) and AI coding tools (Copilot, Cursor). Own the review-specific AI space.
