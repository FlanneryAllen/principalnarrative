What is SOC 2 Type II?

SOC 2 (System and Organization Controls 2) is a security framework developed by the American Institute of CPAs. It's the gold standard for demonstrating that a SaaS company handles customer data securely.

There are two types:

  • Type I: Evaluates security controls at a single point in time
  • Type II: Evaluates security controls over an extended period (typically 6-12 months)

Type II is more rigorous because it demonstrates that security practices are consistently followed, not just documented. Our audit covered a 9-month observation period.

What the Audit Covered

Our SOC 2 audit examined five trust service criteria:

Security

Protection against unauthorized access. This includes our access controls, network security, vulnerability management, and incident response procedures.

Availability

System uptime and performance monitoring. Our 99.9% uptime SLA is backed by redundant infrastructure and automated failover.

Processing Integrity

Ensuring system processing is complete, accurate, and authorized. Every code review follows documented procedures with validation at each step.

Confidentiality

Protection of confidential information. Your code is confidential by default, with strict access controls even within our organization.

Privacy

Personal information handling practices. We collect minimal data and never share or sell customer information.
